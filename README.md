## 🚀 Portfolio Projects

### 🍱 [レンジタイムアシスト](https://masaruookawa.github.io/range-ta-react-app/)
> お弁当を複数まとめて効率よく電子レンジで温めるための計算ツール

- **使用技術**: React / JavaScript / Sass
- **ポイント**: タイマー機能を内蔵。シンプルな操作性を重視したUX設計
- 「限られた休憩時間を最大活用したい」という実体験から生まれたアイデア

### 🌤️ [天気アプリ](https://masaruookawa.github.io/weather-app-react/)
> 現在地または地名から天気情報を取得し、地域に合わせた背景マップを動的表示

- **使用技術**: React / JavaScript / CSS
- **ポイント**: 外部API連携・位置情報取得・動的UI変化

### 🌐 [ポートフォリオサイト](https://masaruookawa.github.io/portfolio/)
> 自己紹介・作品集・お問い合わせを一体化したポートフォリオ

- **使用技術**: HTML / CSS / JavaScript / Vanta.js / EmailJS
- **ポイント**: Vanta.jsで動きのあるTOP画面、EmailJSでお問い合わせ機能を実装

### 🏢 WordPress サイト制作
> フリーランスの方のWeb制作サポートとして、企業サイトを複数実装

- **担当**: デザイン・コーディング・基本フレームワーク構築
